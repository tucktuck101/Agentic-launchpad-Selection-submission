# Launchpad Code Review App: Order Support

This is a small Express and React app for the Agentic Launchpad code review task.

It is intentionally imperfect. Your job is to review it, explain the risks you notice, and choose what you would improve first. You do not need to fix the app.

## What The App Does

The app shows ecommerce orders for a support team. A support agent can:

- View recent orders.
- Filter by order status.
- Mark an order as contacted.
- Add an internal support note.

## Run It

You need Node.js installed.

```bash
npm install
npm run dev
```

The API runs on `http://localhost:4100`.

The React app runs on `http://localhost:5173`.

## Review Prompt

Spend around 20 to 30 minutes reviewing the code.

Focus on:

- Bugs or behaviour that could confuse users.
- Security, privacy, or data-handling risks.
- React state and rendering issues.
- API design and validation.
- What you would change first, and why.

You can read the code without running it. If you do run it, include anything you noticed while testing.
